# Personal Website

A modern, responsive personal website built with Bootstrap 5 and custom styling.

## Description

This is a multi-page personal website featuring a dashboard-style interface with dark purple theme. The website includes pages for overview, wallet management, profile, and contact information.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean and modern interface with smooth animations
- **Search Functionality**: Search across pages and navigate to relevant content
- **Contact Form**: Interactive contact form with submission feedback
- **Dark Theme**: Beautiful dark purple background with white content blocks

## Technologies & Frameworks

- **Bootstrap 5**: Frontend framework for responsive design
- **jQuery**: JavaScript library for DOM manipulation and event handling
- **Bootstrap Icons**: Icon library for UI elements
- **ApexCharts**: Chart library for data visualization (used in wallet page)
- **Roboto Font**: Google Fonts typography
- **Picsum Photos**: Placeholder image service for stock photos

## Pages

1. **Overview** (`index.html`): Dashboard homepage with quick access cards
2. **My Wallet** (`wallet.html`): Transaction history and balance management
3. **Profile** (`profile.html`): Personal information and account details
4. **Contact** (`contact.html`): Contact form and information

## Customization

- Custom CSS styling in `css/tooplate-mini-finance.css`
- Custom JavaScript functionality in `js/custom.js`
- Dark purple color scheme (#4B0082)
- Smooth animations and transitions
